
# 浙大会务接口文档

## 用户签到接口
## 请求方式 : GET

统一URL地址：
	
```
URL/api/v1/order/verify
```


> 参数

测试示例

| 参数 | 类型 | 备注 | 可空 | 测试示例 |
| --- | --- | --- | --- | --- |
| content| string | 二维码扫码内容 | 否 | |
| api_token|string|TOKEN|否



#### 返回成功
```
{
    "message": "签到成功",
    "code": 200,
    "data": ""
}
```

#### 返回失败
```
{
    "message": "你已经签到过了,不用重复签到了～",
    "code": 300,
    "data": ""
}

{
    "message": "没有你的签到记录哦～",
    "code": 300,
    "data": ""
}

```
