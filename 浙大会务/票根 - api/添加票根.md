
# 浙大会务接口文档

## 添加票根接口
## 请求方式 : POST

统一URL地址：
	
```
URL/api/v1/ticket?api_token=___
```


> 参数

测试示例

| 参数 | 类型 | 备注 | 可空 | 测试示例 |
| --- | --- | --- | --- | --- |
| name| string | 票根名称 | 否 | |


#### 返回成功
```
{
    "message": "添加成功",
    "code": 200,
    "data": ""
}
```
#### 返回失败
```
{
    "message": "参数丢失",
    "code": 300,
    "data": ""
}

{
    "message": "数据已存在",
    "code": 300,
    "data": ""
}
```
