
# 浙大会务接口文档

## 更新服务接口
## 请求方式 : PUT

统一URL地址：

```
URL/api/v1/service/put?{service_id}api_token=?
```


> 参数

测试示例

| 参数 | 类型 | 备注 | 可空 | 测试示例 |
| --- | --- | --- | --- | --- |
{"name":"2222"}

#### 返回成功
```
{
    "message": "更新成功",
    "code": 200,
    "data": ""
}
```
#### 返回失败
```
{
    "message": "找不到数据",
    "code": 400,
    "data": ""
}
```
